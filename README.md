The project template of Kotlin Spring Boot

- Spring Boot

# Dependencies
- Spring Boot DevTools
- Spring Web Starter
- Spring Security
- Spring Data JPA
- H2 Database
- PostgreSQL Driver
- Liquibase Migration
- Spring Boot Actuator
